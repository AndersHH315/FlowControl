# FlowControl
## Om projektet
En enkel consol applikation där man får några alternativ på vad programmet kan göra via en huvud meny som ser ut som liknande:

1. "Skriv 1 för att köpa biobiljett", och få ett kvitto på priset.
2. "Skriv 2 för att köpa flera biljetter", för en grupp och få ett kvitto för antal personer och priset.
3. "Skriv 3 och mata sedan in en text". Texten kommer att upprepas 10 gånger på samma rad.
4. "Skriv 4 och Mata in en mening på minst 3 ord". Den kommer efteråt att skriva ut de tredje ordet i meningen.
0. "Skriv 0 för att stänga ner", programmet kommer avslutas.

### Användbarhet
Programmet har input checks ifall användaren skriver in fel, och får ut ett meddelande som hänvisar felet samt vad de bör skriva in för att fortsätta.
